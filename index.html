HTML(Hyper Text Markup Language)

HTML is the standard markup Language for creating web pages. it describes the structure of a web page suing markup.

  • Block Elements:< h1 >, < p >, < ul >, < ol >

  • Inline Elements < i >, < strong >, < img >

  • Attributes : type, src, alt, href

CSS ( Cascading Style Sheets)

CSS is used for styling web pages. it describes how elements should be displayed on the screen.

  • Font properties: font-family, font-size, color

  • Box Model: mardin, padding, border

  • Text properties: text-aligm, line-height,text-decoration.


Property Description
font-family Defines the font of the text
color sets the text color
font-size sets the font size
text-align the alignment for text inside a flexible container
text-decoration specifies the decoration added to text
background-color specifies the background-color of an element
background-img set the background-img for an element
background-repeat sets if/ how a background image will be repeated
bottom sets the element position
